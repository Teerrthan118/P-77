Project On Console

Visual Studio is a IDE that is available on Windows and Mac. It's very easy to set up, you just open https://code.visualstudio.com/Download and download the program.

The console screen tells what is happening in the app, and also if there are any errors in the code. You can open the console screen by right-clicking, clicking on Inspect and going to console. To write on the console, you use the code: console.log
